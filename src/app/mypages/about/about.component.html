About the ToDo App

Hey! Glad you are here. This is my first project while learning Angular and Angular CLI. We can create amazing one web applications with this programming language. So it's a basic ToDo list app made with Angular. We can add our ToDos with the form above handled by Angular formModule, ToDo will be shown on the page immedietly. After adding those will be saved at the client side, we can

  • Create tasks
  • Change status of Todo
  • Remove/Delete tasks whenever we want

It will be done without loading the page (love about this Angular features). I will keep learning and improve this app.
